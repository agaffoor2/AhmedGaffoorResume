UCF Technical Writers Style Guide

Project Description

The UCF Technical Writers Style Guide was a comprehensive, class-wide project for a college course designed to equip new technical writers at the University of Central Florida (UCF) with the parameters needed to produce professional, well-written documents. My primary role involved collaborating on the outlines and drafting content for two key chapters: Chapter Five: Principles of Effective Document Design and Chapter Six: Introduction to Graphics in Technical Documents. This work involved detailing UCF's visual communication standards, including integrating the six core design principles (e.g., grouping, contrast, alignment) and setting technical requirements for all visuals, such as using high-quality images and ensuring brand consistency using the official UCF color palette and logo placement. Ultimately, this project reinforced crucial professional skills in content structuring, utilizing an objective tone, time management, and collaboratively producing highly-readable technical documentation.

Skills Learned / Applied

  • Advanced Research & Evaluation
  • Professional Consistency
  • Technical Communication
